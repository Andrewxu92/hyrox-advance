### 发现的问题

**1. Running 分段提取不稳定**
```typescript
// 当前方法：点击 Running 标签后用 regex 提取
const runningTimes = await page.evaluate(() => {
  const bodyText = document.body.innerText;
```

**问题**: 
- 依赖页面文本格式，容易失效
- 没有 fallback 机制
- 没有验证提取结果

**改进方案**:
```typescript
// 方案 1: 直接查找表格行
$('table tr').each((i, row) => {
```

### 2. 错误处理不足
- 网络超时只返回 null，没有错误信息
- 没有重试机制
- 没有记录失败原因

**改进方案**:
```typescript
interface ScrapeError {
  type: 'network' | 'parse' | 'not_found';
  message: string;
  url: string;
  timestamp: string;
}

async function scrapeWithRetry(url: string, maxRetries = 3): Promise<ScrapedResult | ScrapeError> {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const result = await scrapeFromResultUrl(url);
      if (result) return result;
    } catch (error) {
      if (i === maxRetries - 1) {
        return { type: 'network', message: error.message, url, timestamp: new Date().toISOString() };
      }
      await sleep(1000 * (i + 1)); // 指数退避
    }
  }
}
```

### 3. 没有缓存机制
- 每次抓取都重新请求
- 相同 URL 重复抓取浪费资源

**改进方案**:
```typescript
// 简单的内存缓存
const cache = new Map<string, { data: ScrapedResult; timestamp: number }>();
const CACHE_TTL = 24 * 60 * 60 * 1000; // 24 小时

async function scrapeWithCache(url: string): Promise<ScrapedResult | null> {
  const cached = cache.get(url);
  if (cached && Date.now() - cached.timestamp < CACHE_TTL) {
    return cached.data;
  }
  
  const result = await scrapeFromResultUrl(url);
  if (result) {
    cache.set(url, { data: result, timestamp: Date.now() });
  }
  return result;
}
```

## 📈 Recharts 学习

### 基础图表类型
1. **LineChart** - 趋势图（适合成绩趋势）
2. **BarChart** - 对比图（适合 Station 对比）
3. **RadarChart** - 雷达图（适合能力分析）
4. **AreaChart** - 面积图（适合累计数据）

### 适合 HYROX 的图表

**1. 成绩趋势图 (LineChart)**
```tsx
<LineChart data={raceHistory}>
  <XAxis dataKey="date" />
  <YAxis />
  <Tooltip />
  <Line type="monotone" dataKey="totalTime" stroke="#8884d8" />
</LineChart>
```

**2. Station 对比雷达图 (RadarChart)**
```tsx
<RadarChart data={stationTimes}>
  <PolarAngleAxis dataKey="station" />
  <PolarRadiusAxis />
  <Radar name="本次" dataKey="current" stroke="#8884d8" />
  <Radar name="最佳" dataKey="personalBest" stroke="#82ca9d" />
</RadarChart>
```

**3. 分段对比柱状图 (BarChart)**
```tsx
<BarChart data={runSplits}>
  <XAxis dataKey="runNumber" />
  <YAxis />
  <Bar dataKey="time" fill="#8884d8" />
</BarChart>
```
