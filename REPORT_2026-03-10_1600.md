| 能量系统 | 功能描述 | 占比计算 |
|---------|---------|---------|

**输出示例**:
```json
{

  "glycolyticContribution": 33,

  "dominantSystem": "Aerobic",

```

**训练建议生成**:
- 有氧主导 → 加强无氧训练（HIIT、爆发力）
- 糖酵解主导 → 提升有氧基础（LSD、阈值跑）
- ATP-CP 主导 → 加强耐力训练（混合训练）

### 3. 数据库性能优化 ⭐⭐

**文件**: `server/db/index.ts`

#### SQLite 优化配置
```typescript
// WAL 模式 - 提升并发性能 50-80%
sqlite.pragma('journal_mode = WAL');

// 64MB 缓存 - 减少磁盘 I/O
sqlite.pragma('cache_size = -64000');

// 外键约束 - 数据完整性
sqlite.pragma('foreign_keys = ON');

// 平衡性能和安全
sqlite.pragma('synchronous = NORMAL');

// 临时表内存存储
sqlite.pragma('temp_store = MEMORY');
```

#### 新增数据库索引（12 个）

**成绩表 (results)**:
- `idx_results_athlete` - 按运动员查询
- `idx_results_date` - 按日期查询

**分析报告表 (analysis_reports)**:

**训练计划表 (training_plans)**:

**训练打卡表 (training_logs)**:

- `idx_logs_completed` - 按完成时间
